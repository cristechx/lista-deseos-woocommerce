== Descripción ==

Lista de Deseos para WooCommerce es un plugin completamente personalizable que permite a los usuarios guardar sus productos favoritos de manera intuitiva y atractiva.

=== Características Principales ===

* **Añadir/Eliminar Productos**: 
  - Botón de un clic para añadir o quitar productos de la lista de deseos
  - Funciona en páginas de producto y tienda
  - Totalmente personalizable

* **Diseño Responsivo**: 
  - 100% adaptable a dispositivos móviles
  - Funciona en smartphones, tablets y escritorio
  - Diseño moderno y elegante

* **Personalización Avanzada**:
  - Selector de iconos (Corazón, Estrella, Marcador)
  - Personalización de color
  - Múltiples estilos de botón
  - Soporte para modo oscuro

* **Funcionalidades**:
  - Múltiples listas de deseos
  - Compartir lista de deseos
  - Añadir notas a productos
  - Estadísticas detalladas
  - Importación/Exportación de configuraciones

* **Rendimiento**:
  - Optimizado para velocidad
  - Mínimo impacto en la carga del sitio
  - Código ligero y eficiente

== Instalación ==

1. Instalar el plugin:
   - Opción A: Instalar desde el repositorio de WordPress
   - Opción B: Subir el archivo ZIP desde el panel de WordPress
   - Opción C: Subir los archivos via FTP a la carpeta `/wp-content/plugins/`

2. Activar el plugin:
   - Ir a Plugins
   - Buscar "Lista de Deseos para WooCommerce"
   - Hacer clic en "Activar"

3. Configurar el plugin:
   - Navegar a "WooCommerce" > "Lista de Deseos"
   - Ajustar configuraciones según necesidad

=== Requisitos ===

* WordPress 5.0 o superior

== Uso ==

=== Añadir a Lista de Deseos ===
1. Navegar a un producto
2. Hacer clic en el botón "Añadir a Lista de Deseos"
3. El producto se guarda automáticamente

=== Gestionar Lista de Deseos ===
* Acceder desde "Mi Cuenta" > "Lista de Deseos"
* Eliminar productos individuales
* Limpiar lista completa
* Crear múltiples listas

=== Shortcodes ===
* `[ldw_wishlist_button]`: Mostrar botón de lista de deseos
* `[ldw_wishlist_page]`: Mostrar página de lista de deseos
